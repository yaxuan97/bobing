# bobing
# 博饼（跋饼），是起源于厦门的中秋传统活动随之传播至闽南地区，始于清初，是一种独特的月饼文化，也是闽南人对历史的一种传承。相传，中秋博饼，是郑成功屯兵鼓浪屿时为解士兵的中秋相思之情、激励鼓舞士气而发明的。于是，一代一代传下来，就成了如今闽南地区独具特色的民间习俗。
